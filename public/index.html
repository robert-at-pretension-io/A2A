A2A Agent Registry

MACHINE-TO-MACHINE A service for AI agents to find other AI agents

What is this?

This is a registry for A2A agents to discover each other. It stores information about available agents and their capabilities, acting as a centralized directory for machine-to-machine communication.

How Agents Use This Registry

1. Register an Agent

Send any message containing your agent's URL to register it:

POST repository-agent.com
{
  "jsonrpc": "2.0",
  "id": "request-123",
  "method": "tasks/send",
  "params": {
    "id": "task-456",
    "message": {
      "role": "user",
      "parts": [{
        "type": "text",
        "text": "Register my agent at https://my-agent.example.com"
      }]
    }
  }
}

2. Get List of Agents

Request the registry for a list of all known agents:

POST repository-agent.com
{
  "jsonrpc": "2.0",
  "id": "request-789",
  "method": "tasks/send",
  "params": {
    "id": "task-abc",
    "message": {
      "role": "user",
      "parts": [{
        "type": "text",
        "text": "list agents"
      }]
    }
  }
}

How It Works

The registry automatically:

  • Extracts agent URLs from messages
  • Fetches agent information (AgentCards) from the URLs
  • Stores capabilities in a searchable directory
  • Responds to queries from other agents

This enables a network of specialized agents to discover and collaborate with each other.
